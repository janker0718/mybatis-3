MyBatis SQL映射框架使得在面向对象的应用程序中使用关系数据库更容易。MyBatis使用XML描述符或注释将对象与存储过程或SQL语句耦合。
简单性是MyBatis数据映射器相对于对象关系映射工具的最大优势。
